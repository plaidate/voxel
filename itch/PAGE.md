# Voxel — itch.io page copy

## Tagline (<=80 chars)

Ten destructible 1-bit voxel games for Playdate — dig, lob, carve, climb.

## Description

Voxel is a collection of ten original games for the Playdate, all built on
one tiny Voxatron-style engine: a solid 3D room of dither-shaded voxels in
a 3/4 projection — and nearly every voxel is destructible. Shots blast real
craters, tunnels actually tunnel, slime genuinely floods the room, and the
arena you finish on never looks like the one you started on.

Each game earns its third dimension a different way. Carve a wave-survival
arena to bits in Rubble, spring between crumbling columns above rising
slime in Crumble, duel an artillery AI over a ridge in Lob, wall in your
keep Rampart-style in Bulwark, shepherd lemming-sheep in Herd, sink fossil
shafts in Excavate, roll a marble down a heightfield in Marble, sumo-shove
brutes off a ziggurat in Summit, bomb through dungeon walls in Vault, and
fly the classic VoxelSpace terrain renderer in Voxelspace. The crank does
something distinct in every one.

## Features

- Ten complete games on one shared voxel engine — every one crank-first
- Fully destructible terrain: spherical carving, cave-ins, real craters
- Terrain that grows too: flooding slime, buildable walls, eroding terraces
- A 96x64x16 voxel room, dither-shaded for the 1-bit screen, at 30 fps
- Turn-based artillery with wind and a bracketing AI opponent
- A five-room dungeon crawl with keys, bombs, pots and a saved run
- A bonus tech demo: the Comanche VoxelSpace renderer, crank-steered
- Free and open source (MIT) — the engine is yours to build on

## Controls (summary)

- D-pad — move / steer / aim, game by game
- A — the main verb: fire, jump, place, dig, shove
- B — the secondary verb: tools, bombs, cancel
- Crank — a different dial in every game: aim, power, release rate,
  spin-up, boost, steering

Full per-game controls are in each game's section of MANUAL.md.

## Install

1. Download the `.pdx.zip` files you want from the Releases page (or grab
   the `.pdx` folders from `dist/` in the repo).
2. On device: sideload at https://play.date/account/sideload/ — upload the
   zip, then download it to your Playdate from Settings → Games.
3. In the Playdate Simulator: unzip and open the `.pdx` (or drop it onto
   the Simulator window). No developer toolchain needed either way.
